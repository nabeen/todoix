# タスク: 032 カスタム日付入力コンポーネント実装

## ゴール

- ユーザーが独自の`from`/`to`日付を入力できるInkコンポーネントを実装する

## 作業内容

- `packages/ui`に日付入力用コンポーネント（例: `DateRangeInput`）を追加し、ISO形式の文字列やカレンダー風入力を受け付ける
- フォーカス移動・バリデーション・エラーメッセージ表示を実装する
- 入力確定時に状態モデルへ日付レンジを通知する

## 完了条件

- キーボード操作のみでfrom/toを編集・確定できる
- 不正な日付フォーマットやfrom>to時にエラー表示が出る
